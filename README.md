# RETO_SEM10
Elaboración de dos listas con diferentes longitudes, eliminando elementos repetidos
Definición de dos funciones, la primera pregunta por el numero de ingresos a ésta,creando la lista, a gregando los elementos i + 1.
La segunda función elimina los elementos repetidos entre ambas listas, de la primera o lista1.
